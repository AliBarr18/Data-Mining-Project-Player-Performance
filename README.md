# Data-Mining-Project-Player-Performance 
Problem Statement
The goal of this project is to predict a player’s in‑game performance based on historical statistics, physical attributes, and gameplay metrics.
This prediction helps coaches, analysts, and teams make informed decisions about player development, game strategy, and roster management.

Depending on the dataset, “performance” may refer to:

- Total points scored

- Efficiency rating

- Fantasy points

- Overall performance index

- Or another measurable performance metric

Your model will translate raw player data into a predictive performance outcome.

2. Target Variable
Target Variable: Player Performance Metric  
(Example: Points, Efficiency Rating, Fantasy Score, etc.)

The exact variable depends on the dataset you selected, but it must be:

- Quantifiable

- Predictable from player features

- Meaningful for evaluating performance

3. Use Case / Domain Context
This project sits in the sports analytics domain, a rapidly growing field used by:

- Professional teams

- College programs

- Scouts

- Fantasy sports platforms

- Sports betting models

Predicting player performance supports:

- Game planning

- Player comparison

- Training focus areas

- Recruitment decisions

- Performance forecasting

Sports organizations increasingly rely on data-driven insights, making this type of model highly relevant.

4. Dataset Selection & Justification
Dataset: Player Performance Dataset (the one stored in your repo)

Why this dataset works:

- Contains real player statistics

Includes predictive features such as:

- Physical attributes (height, weight, position)

- Game statistics (minutes, attempts, rebounds, assists, etc.)

- Historical performance trends

Has a clear target variable representing performance

Suitable for regression or classification modeling

Clean enough for preprocessing but still realistic for a data mining project

This dataset directly supports the goal of predicting player performance.

5. Potential Value of the Model (Decision-Making Impact)
A strong predictive model can help:

- Coaches identify which players are likely to perform well in upcoming games

- Teams optimize lineups and rotations

- Scouts evaluate player potential

- Analysts forecast season-long performance trends

- Fantasy sports users make better roster decisions

The model provides actionable insights that improve decision-making across multiple areas of sports analytics.
